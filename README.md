# Calculadora de Divisão - JavaFX

Este é um projeto de estudo desenvolvido para praticar conceitos de interfaces gráficas com **JavaFX** e arquitetura de software. A aplicação realiza cálculos matemáticos simples com validação de dados.

## Funcionalidades

- **Cálculo de Divisão:** Exibe o resultado exato da divisão entre dois números.
- **Cálculo de Resto (Módulo):** Exibe o resto da divisão (útil para verificar paridade).
- **Tratamento de Erros:** Utilização de blocos `try-catch` para capturar exceções (`NumberFormatException`), impedindo que o programa trave caso o usuário digite letras ou símbolos.
- **Arquitetura MVC:** Separação clara entre a interface (`.fxml`) e a lógica de controle (`Controller.java`).

## Tecnologias Utilizadas

- **Java** (Linguagem base)
- **JavaFX** (Interface Gráfica)
- **Maven** (Gerenciamento de dependências)
- **VS Code** (IDE utilizada)

## Como rodar o projeto

1. Clone este repositório.
2. Abra a pasta do projeto no VS Code ou IntelliJ.
3. Aguarde o Maven carregar as dependências.
4. Execute o arquivo `App.java`.
